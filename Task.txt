# React Notes (26-02-2026)
1️⃣ Component
A component is a reusable part of the UI.
Written with first letter capitalized.
JSX components are enclosed in <> </> (or <div></div> for multiple tags).

Example:
function App() {
  return (
    <div>
      <h1>Div is used if we have to wrap more than one tag</h1>
      <h2>This div works like curly brackets in JS for grouping elements</h2>
    </div>
  );
}

2️⃣ Function

A function is a block of code to perform a task.
Written in lowercase letters.
Enclosed in {add()} when used inside a component.

Example – 

Normal Function for Adding Two Numbers
function add(a, b) {
  return a + b;
}
Calling the Function Inside a Component
function App() {
  const result = add(5, 3); // calling the function
  return (
    <>
      <h1>This is a simple sentence.</h1>
      <h2>Addition Result: {result}</h2>
    </>
  );
}

      An arrow function is a shorter way to write a function in JavaScript.

              const greet = name => `Hello, ${name}!`;

          // Calling the  function
